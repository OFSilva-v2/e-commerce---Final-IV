# e-Footwear Outlet - Site de E-commerce

Site profissional para loja de calçados e bolsas com funcionalidades completas de e-commerce.

## Características
Desenvolvido um SITE para Empresa e-Footwear Outlet. Acessando através do Android 

- Catálogo de produtos com 3 linhas (Tênis, Sapatos Sociais, Bolsas)
- 4 produtos por linha (total: 12 produtos)
- Sistema de carrinho de compras
- Checkout com múltiplas formas de pagamento
- Área do cliente com histórico de pedidos
- FAQ interativo
- Formulário de contato
- Design responsivo
- Logo centralizada com links rápidos
